Even-Odd Checker App
This is a simple Android application built using Java and XML. It allows users to input a number and instantly check if it is Even or Odd.

Features
Interactive User Interface.
Real-time number verification.
Lightweight and easy to use.

How to use
Launch the app.
Enter any integer in the input field.
Tap the "Check" button to see the result.

Tech Stack
Language: Java
UI Design: XML
IDE: Android Studio
